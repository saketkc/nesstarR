# List datasets in a parsed NESSTAR container

List datasets in a parsed NESSTAR container

## Usage

``` r
nesstar_datasets(x)
```

## Arguments

- x:

  A `nesstar_binary` object from
  [`nesstar_parse`](https://saketkc.github.io/nesstarR/reference/nesstar_parse.md).

## Value

A data frame with columns `dataset_number`, `row_count`,
`variable_count`.
