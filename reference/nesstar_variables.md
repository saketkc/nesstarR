# List variables in one dataset

List variables in one dataset

## Usage

``` r
nesstar_variables(x, dataset_number)
```

## Arguments

- x:

  A `nesstar_binary` object.

- dataset_number:

  Integer dataset number (as shown by
  [`nesstar_datasets`](https://saketkc.github.io/nesstarR/reference/nesstar_datasets.md)).

## Value

A data frame with one row per variable describing its storage
properties.
